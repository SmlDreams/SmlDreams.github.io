Bienvenue sur la page Infrastructure d'acfri

Au cours de mon Stage Acfri m'a demandé de réaliser le plan de l'infrastructure d'acfri, voici un aperçu de ce que j'ai réalisé.

Pour détailler cette infrastructure, il y a plusieurs éléments à prendre en compte, voici les principaux éléments:
    - Un serveur principal ( physique ) :
         - une machine virtuelle pour l'active directory
         - une machine virtuelle pour le systeme de fichier
         - une machine virtuelle pour la production
         - une machine virtuelle pour l'accès à distance
    - 17 postes de travail :
         - 10 windows 11
         - 5 windows 10
         - 2 windows 7
    - 1 stockage des sauvegardes
    - 4 ordinateurs de contrôles pour les machines de production
    - 2 switchs :
         - un switch principale pour les servers et les postes de travail
         - un switch pour les machines de production
    - 1 firewall
    - 2 box de connection sans fil
    - 2 fibres optiques

Pour la partie en haut à droite du schéma, il s'agit de la partie externe de l'entreprise, voici les éléments:
     - le prestataire pour le site internet
     - le prestataire pour la partie sauvegarde dans le cloud
